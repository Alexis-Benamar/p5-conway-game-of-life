> _Cells! They live, they die. It's the cycle of life_

## Setup

No setup required, just open the `index.html` file in your browser.

## Controls

* **S**: stop / resume
* **R**: random grid
* **E**: empty grid
* **O**: cell style
* **T**: dark / white mode
* **Mouse Click**: cell toggle
